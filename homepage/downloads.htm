Download Area
 

The framework is divided into a main program that has full functionality and add-ons for additional features, as for example a package with mex-files to replace not vectorized loops to increase performance. To install any of the following add-ons, simply extract all files from the desired add-on zip file to the root directory of the FFW main program.

The documentation for the framework is also seperated into two parts. The first is a PDF file which is maintained along with changes in the FFW. It can be downloaded below. The second part of the documentation is a HTML based function-reference database created with M2HTML. This database is enclosed in the FFW main program.

Diploma Thesis: A mixed finite element method for a nonlinear Optimal-Design problem
This thesis describes the modelling of a nonlinear optimal design problem and its discretization using the Raviart Thomas element. Different reliable error estimators are proposed to refine adaptively the initial mesh. The implementation is done in the OpenFFW framework and is available in the OptimalDesign branch. This thesis is written in german.
