Features
 

The FFW uses a global data structure containing all computed data, including complete mesh information like edge enumeration, normals, tangents, etc.. Furthermore the framework has full script ability for automatic computation with different parameters. Thus it provides a general framework for output routines to analyse the finite element methods and its results.

There are already various test problems to illustrate performance of adaptivity and test correctness. Furthermore, own problems can be easily created using predefined templates. There is also a template where only a domain and an exact solution is to be specified, and all other data is calculated through MATLAB's symbolic differentiation toolbox.

The FFW currently features the following finite element methods:

  • P1-FEM, P2-FEM, P3-FEM a standard conforming discretization for elliptic PDE's,
  • CR-FEM, a non-conforming discretization for elliptic PDE's,
  • RT0-P0-MFEM, a mixed FEM for elliptic PDE's,
  • Optimal Design P1-FEM, P2-FEM, standard conforming discretization for a nonlinear optimal design problem (extra branch: OptimalDesign)
  • Optimal Design RT0-P0-MFEM, a mixed FEM for a nonlinear optimal design problem (extra branch: OptimalDesign)
  • P1P1, a standard conforming discretization for elasticity problems,
  • P1CR, a non-conforming and locking free discretization for elasticity problems,
  • Arnold Winther, a mixed, higher order, locking free FEM for elasticity problems.

Adaptivity algorithms provided:

  • A newest vertex bisection like algorithm for edge oriented mesh refinement that produces shape regular, nested triangulations with no hanging nodes and allows for the H^1 stability of the L^2 projection,
  • Graded meshes using the above algorithm,
  • Reliable and efficient a posteriori error estimators for almost all of the above methods,
  • Two marking strategies, maximum and bulk, controlling the mesh refinement based on a posteriori error estimators.

Further features:

  • A simple interface for numerical integration to easily compute boundary integrals, error norms, etc.,
  • A simple reference multigrid implementation for P1-FEM.
