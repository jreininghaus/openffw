matMul

SYNOPSIS

function val = matMul(A,B)

DESCRIPTION 

 For given 3-dimensional matrices A ( dim(A) = [n m k] ) and
 B ( dim(B) = [m l k] ) matrixMultiplication computes the elementwise
 matrix product A(k)*B(k)
