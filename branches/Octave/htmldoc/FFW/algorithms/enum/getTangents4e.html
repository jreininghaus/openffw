getTangents4e

SYNOPSIS

function tangents4e = getTangents4e(c4n,n4e,length4ed,ed4e)

DESCRIPTION 

 tangents4e
 The function getNormnals4e returns a [3 2 nrElems] matrix. the input is 
 c4n, n4e and length4ed and ed4e produced by the functions getLength4ed and getEd4e.
 The three dimensional matrix contains in the j-th 3 times 2 matrix the coordinates
 of the unit tangents in counter clockwise direction for the three edges of element j,
 i.e. the direction of the edges. The order of the three rows corresponds two the order
 in ed4e.
